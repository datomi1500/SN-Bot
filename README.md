# SN - Bot

**SN-Bot** Es un bot de Discord potente y repleto de funciones, diseñado para mejorar tu experiencia en el servidor con herramientas de moderación completas, sistemas económicos atractivos, funciones prácticas y mucho más. Desarrollado con la versión moderna de Discord.js v14 y PostgreSQL para garantizar un rendimiento óptimo y la persistencia de los datos.

## Resumen de características

SN-Bot ofrece un conjunto completo de herramientas para la gestión de servidores de Discord y la participación de la comunidad:

<table>
<tr>
<td width="50%" valign="top">

### Moderación y administración
- **Acciones masivas** - Funciones para bloquear o expulsar a usuarios de forma masiva
- **Notas del usuario** - Mantener registros detallados de la moderación
- **Gestión de casos** - Ver y realizar un seguimiento de todas las acciones de moderación

### Sistema económico
- **Tienda e inventario** - Comprar y gestionar artículos
- **Apuestas** - Arriesgarlo todo a cambio de recompensas
- **Sistema de pagos** - Transferir dinero entre usuarios

### Diversión y entretenimiento
- **Datos curiosos** - Aprende algo nuevo
- **Cartel de «Se busca»** - Crea divertidos carteles de «Se busca»
- **Inversión de texto** - Invierte cualquier texto

### Sistema avanzado de tickets
- **Reclamaciones y prioridad** - Gestión de tickets por parte del personal
- **Límites de tickets** - Prevención del spam
- **Sistema de historiales** - Guardar el historial de tickets

### Estadísticas del servidor
- **Contador de miembros**: canales con recuento de miembros en tiempo real
- **Contadores de voz**: seguimiento de las estadísticas de voz
- **Actualizaciones dinámicas**: actualizaciones de los canales en tiempo real

### Roles de reacción
- **Asignación de roles**: roles que cada uno puede asignarse a sí mismo
- **Selección de emojis**: sistema basado en reacciones
- **Compatibilidad con múltiples roles**: múltiples opciones de roles

## Intenciones necesarias para el bot
SN-Bot requiere las siguientes intenciones de Discord:
- **Guilds**
- **Guild Messages**
- **Message Content**
- **Guild Members**
- **Guild Message Reactions**
- **Guild Voice States**
- **Direct Messages**
- **Bot**
- **Applications.commands**

### Permisos necesarios
- **View Channels**
- **Send Messages**
- **Embed Links**
- **Attach Files**
- **Read Message History**
- **Manage Messages**
- **Manage Channels**
- **Manage Roles**
- **Kick Members**
- **Manage Messages**
- **Ban Members**
- **Moderate Members**
- **Connect**
